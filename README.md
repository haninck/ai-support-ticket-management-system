# AI Support Ticket Management System

## Overview

AI Support Ticket Management System is a web-based application that helps organizations manage customer support tickets efficiently. The system uses Machine Learning to automatically classify support tickets, assign priorities, and streamline ticket management.

## Features

### Customer Module

* Customer Signup and Login
* Submit Support Tickets
* View Submitted Tickets
* Track Ticket Status
* View Assignment Details
* View Support Notes

### Admin & Agent Module

* Secure Login
* Ticket Dashboard
* Ticket Search and Filtering
* Assign Tickets to Agents
* Add Internal Notes
* Update Ticket Status
* View Ticket Creator Information

### AI Features

* Automatic Ticket Classification
* Category Prediction
* Confidence Score Calculation
* Priority Prediction
* Multi-Category Detection
* Suggested Response Generation

## Technology Stack

## Workflow

Customer → Submit Ticket → AI Classification → Admin Review → Agent Assignment → Status Updates → Customer Tracking
